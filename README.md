# PyTorch-Unet
PyTorch1.0 Implementation of Unet

- Vanilla Unet
- Unet with ResNet encoder
  - ResNet18
  - ResNet34
  - ResNet50
  - ResNet101
  - ResNet152
- Unet with VGG encoder
  - VGG13 with BatchNorm
  - Very easy to be modified into other VGG architectures
